# RESPUESTAS

## Decisiones técnicas

- Se usó arquitectura por módulos (routes/service/repository) para separar responsabilidades.
- Se aplicó `zod` para validar payloads y query params en endpoints.
- Se mantuvo `pg` como cliente SQL directo para mantener la prueba enfocada en backend base.

## Uso de IA

Este entregable fue asistido por IA para acelerar la creación de estructura base y documentación.
